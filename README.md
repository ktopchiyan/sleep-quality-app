# Sleep Quality Predictor
An interactive web application that predicts sleep quality (0–10) based on health and lifestyle factors.  
Built with **Python, Pandas, Scikit-learn, Streamlit**, and deployed via **GitHub + Streamlit Cloud**.  

##Project Structure 
- `app.py` – Streamlit app script  
- `sleep_quality_model.pkl` – Trained Random Forest model  
- `feature_names.pkl` – Feature names from training data  
- `category_options.pkl` – Encoded categorical options  
- `requirements.txt` – Dependencies for deployment
